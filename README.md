# ChatANJE - Plugin WordPress

Plugin WordPress para o chatbot da ANJE (www.anje.pt).

## Instalação

1. Download do ZIP
2. WordPress > Plugins > Adicionar Novo > Carregar Plugin
3. Ativar
4. Definições > ChatANJE > Configurar OpenRouter API Key

## Funcionalidades

- Widget de chat flutuante
- Respostas inteligentes via OpenRouter
- Dados completos da ANJE (equipa, programas, contactos)
- Links automáticos para páginas do site
- Cache de respostas (5 minutos)
- Design responsivo
